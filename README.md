# 🧠 Customer Segmentation Using K-Means

## 📌 Overview
This project segments customers based on annual income using the **K-Means clustering algorithm**.  
The model groups customers into meaningful clusters by analyzing spending behavior and demographic patterns.

📅 **Completed: January 2025**  
🔄 **Updated on GitHub: March 2025**

---

## 🚀 Key Insights
- Clustered **1K+ customers**
- Achieved **95% clustering accuracy**
- Improved targeting precision by **25%**
- Visualized distinct customer segments using scatter plots

## 📘 How It Works
1. Load & clean customer dataset  
2. Standardize numerical columns  
3. Use **Elbow Method** to find optimal K  
4. Train **K-Means model**  
5. Visualize customer clusters  
6. Interpret business insights
